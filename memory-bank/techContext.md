# Tech Context

## Core Stack

- **Runtime:** Bun
- **Package Manager:** Bun (Always use `bun install`, `bun run`, `bunx` instead of `npm`/`npx`)
- **Language:** TypeScript
- **Build Tool:** Vite
- **Backend/Database:** Convex (BaaS)
- **UI Framework:** React
- **Styling:** Tailwind CSS + shadcn/ui
- **Routing:** TanStack Router

## Libraries & Tools

- `react-hook-form`: Form state management.
- `zod`: Schema validation.
- `@tanstack/react-router`: Routing.
- `lucide-react`: Icons.
- `cmdk`: Command palette component (used for "Add Album" search).

## External APIs

- **MusicBrainz:** For album search and metadata auto-fill.
